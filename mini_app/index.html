📖 Track24.ru — бесплатно

Покупатель нажимает ссылку в боте → переходит на track24.ru → вводит трек-номер → видит статус.

Что нужно сделать вам:
1. Зайдите на track24.ru
2. Зарегистрируйтесь (бесплатно)
3. В разделе «Посылки» добавьте трек-номер вручную
4. Сайт будет автоматически обновлять статус

Покупатель переходит по ссылке и видит статус самостоятельно.
📖 17Track.net — платный API

Покупатель нажимает ссылку → переходит на 17track.net → вводит трек-номер вручную.

Как подключить автоматический статус:
1. Зайдите на 17track.net → Sign Up
2. Войдите → перейдите в раздел Developer → API
3. Создайте приложение → получите API-ключ
4. Выберите тариф (от $0.003 за запрос)
5. Вставьте API-ключ в настройки бота

Без API: покупатель просто вводит трек на сайте вручную — это бесплатно.
