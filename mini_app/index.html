📖 Track24.ru — бесплатно

Покупатель нажимает ссылку → переходит на track24.ru → вводит трек-номер → видит статус.

Что нужно вам:
1. Зайдите на track24.ru → зарегистрируйтесь (обычная почта)
2. В разделе «Посылки» добавьте трек-номер вручную
3. Сайт автоматически обновляет статус
📖 Track24 API — платный автостатус

Бот сам показывает статус посылки прямо в чате без перехода на сайт.

Как получить API:
1. Зайдите на track24.ru → зарегистрируйтесь
2. Перейдите в раздел API или напишите в поддержку
3. Получите ключ и вставьте ниже
4. Стоимость уточняйте на сайте
